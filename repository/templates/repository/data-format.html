Satellite Observation Data Format

To enable systematic compilation and curation of observations from various sources, the IAU CPS SatHub requires submissions to adhere to the following format.

The required file type for submission is a Comma Separated Value (CSV) text file containing a set of required columns ordered and named as below. Any or all optional columns may also be included in the order listed. A sample file that meets all formatting requirements may be downloaded below. Other observation formats are not currently supported.

Please note that by sharing observations with the IAU CPS SatHub, you agree to follow the IAU CPS Member Collaboration Agreement. This entails that observations you submitted can be published, or made otherwise available by third parties as long as you are credited and your data is properly attributed.


Required columns

satellite_name
Name of satellite, string, corresponds to https://www.space-track.org/#catalog SATNAME, e.g., "METEOR 1-17 DEB"
norad_cat_id
NORAD catalog identifier, integer, corresponds to https://www.space-track.org/#catalog NORAD CAT ID, e.g., "57957"
observation_time_utc
Mid exposure time of observation in coordinated universal time (UTC), ISO 8601 format (string), e.g., "2023-09-26T23:59:59.123Z"
observation_time_uncertainty_sec
Uncertainty in the observation time in seconds, floating point, e.g., .02
apparent_magnitude
Brightness of the satellite as observed/measured in magnitudes, floating point, e.g., "7.53"
apparent_magnitude_uncertainty
Uncertainty in the brightness of the satellite in magnitude, floating point, e.g., "0.14"
observer_latitude_deg
Latitude of the observing location (GPS/WGS84/EPSG Coordinate Reference System Code 4326) in decimal degrees (DD), signed floating point between -90 and 90, e.g. "-30.2447392" (equivalent to 30°14'40.7"S).
observer_longitude_deg
Longitude of the observing location (GPS/WGS84/EPSG Coordinate Reference System Code 4326) in decimal degrees (DD), signed floating point between -180 and 180, e.g., "-70.7477219" (equivalent to 70°44'57.9"W)
observer_altitude_m
Altitude of the observing location above sea level (GPS/WGS84/EPSG Coordinate Reference System Code 4326) in meters, signed floating point, e.g., "2663" (equivalent to 8,737 ft)
instrument
Name or other identifying information about the telescope, instrument, and/or other hardware that was used for the observation, string, e.g., DECam on the Blanco 4m
observing_mode
Observation hardware / detector: e.g. "VISUAL", "BINOCULARS", "CCD", "CMOS", or "OTHER" if not listed
observing_filter
Filter/Bandpass used in observation: e.g., SDSS g. Use "CLEAR" if observing_mode is visual.
observer_orcid
List of ORCIDs for all people who should be credited for the observation. At least one is required; getting an ORCID is free and easy at orcid.org.
observer_email
Valid email address for the observer or point of contact.

Optional columns

satellite_right_ascension_deg
Right Ascension of satellites in deg, measured with respect to Gaia DR3 astrometric catalog stars, floating point between 0 and 360, e.g., "359.12345678"
satellite_declination_deg
Declination of satellites in deg, measured with respect to Gaia DR3 astrometric catalog stars, floating point between -90 and 90, e.g., "-40.12345678"
satellite_ra_dec_uncertainty_deg
Uncertainty matrix of the above values, floating point, e.g., "0.01, 0.01, 0.01, 0.01, 0.01, 0.01" to be read as a 2x3 matrix
range_to_satellite_km
Measured distance between observer and satellite at time of observation in kilometers, floating point e.g., "560.12345678"
range_to_satellite_uncertainty_km
Uncertainty of the above value, floating point, e.g., 0.01
range_rate_of_satellite_km_per_sec
Measured change in distance between observer and satellite at time of observation in kilometers per second, floating point e.g., "3.12345678"
range_rate_of_satellite_uncertainty_km_per_sec
Uncertainty of the above value, floating point, e.g., 0.01
comments
Any comments that would help identify potential issues with the measurement, e.g. "cloudy conditions", "few background stars to compare", "strong lights in the background", etc.
data_archive_link
Link to data archive containing the original data
