Getting Started

  1. Familiarize yourself with the data format: Download the sample observation CSV file to understand the required and optional fields. Make sure to follow the correct column ordering as outlined in the sample file.
  2. Prepare your data: Organize your satellite brightness and position measurements in a CSV file format, using the SCORE-specific columns. Leave blank any fields that are not applicable, and avoid using "n/a", none, 0, or similar values.
  3. Generate a CSV file from individual observations: If you don't have a CSV file ready, use the web tool to generate one from individual observations.
  4. Review and validate your data: Double-check your data for errors and ensure it meets the SCORE format requirements.
  5. Upload your data: Submit your CSV file to SCORE, and the system will process and store your data. If there are errors, you'll receive an error message, but correctly formatted observations will be saved.
  6. Cite SCORE in your research: If you use SCORE data in your research, be sure to include the required acknowledgement paragraph in your publication.
